# ZENITHIST ウェブサイト

## GitHubへのアップロード手順

1. `zenithist.github.io` のリポジトリを開く
2. 既存のファイルを全て削除
3. 以下のファイルをアップロード
   - `index.html`
   - `style.css`
   - `images/` フォルダ（下記参照）

## 画像ファイルの準備

`images/` フォルダを作成して以下を配置：

| ファイル名 | 内容 |
|-----------|------|
| `campaign_course.jpg` | コースのキャンペーン画像（2人・後ろ姿） |
| `product_black.jpg` | BLACKセットアップの商品写真 |
| `product_stone.jpg` | STONEセットアップの商品写真 |
| `logo.png` | ZNエンブレムロゴ（黒背景・白ロゴ） |

## BASEのURLを設定

`index.html` 内の `YOUR_BASE_URL` を実際のBASEショップURLに変更する（2箇所）
